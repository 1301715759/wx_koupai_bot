# 微信扣排机器人
## 项目简介
基于千寻框架callback接口实现消息接收
采用python+redis(8.0以上版本)作为中间件+sqlite本地持久化存储
使用celery异步任务队列处理定时任务队列（后续可能会更新无需celery的版本，因为原计划是处理超多消息队列的情况才采用celery队列去管理控制）
